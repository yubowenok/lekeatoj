Problem A - Multiplication
Time Limit:3000MS Memory Limit:64MB
Description
This is a happy test problem. Given two non-negative integers, compute their multiplication.
Input
Each test case has two integers, a and b.
Output
Output the multiplication as a single integer.
Test Specification
  • Small: 0 ≤ a,b ≤ 100
  • Medium: 0 ≤ a,b ≤ 109
  • Large: 0 ≤ a,b ≤ 10100
  • Challenge: 0 ≤ a,b ≤ 10100000
Sample(s)
Input #1
3 5
Output #1
15
Input #2
1 100
Output #2
100
Input #3
123456789 987654321
Output #3
121932631112635269
